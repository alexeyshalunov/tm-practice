History Object
    History Properties: 
    
    length — кількість елементів у списку відвіданих сторінок.
    
    
    History Methods:
    
    • back() — завантажує попередню сторінку зі списку;
    • forward() — завантажує наступну сторінку зі списку;
    • gо() — завантажує сторінку з відносним номером n зі списку ( від 0 до history.length-1) або із зазначеною Url-адресою

    if (window.history.length>4) {
      window.history.go(5);
    }

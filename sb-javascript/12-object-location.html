Location Object
    Location properties:
    
    • href — повна Url-адреса у вигляді рядка;
    • hash — рядок, що знаходиться в URL за символом #, за допомогою  якого вказується, до якого анкера слід переміститися при  завантаженні документа.;
    • host — частина URL “хост.порт”; значення порту міститься лише тоді, коли воно явно було зазначено в Url-адресі;
    • hostname — частина URL “хост”;
    • pathname — шлях до об'єкта або файлу, що перебуває після третього “/”;
    • port — номер порту URL;
    • protocol — початкова частина, що визначає протокол, за якою йде двокрапка, наприклад “http:”;
    • search — рядок запиту або дані URL після знака “?”.
    
    Location Methods:
    
    • assign() — завантажує іншу сторінку; цей метод еквівалентний зміні властивості window.location.href;
    • reload() — обновляє поточну сторінку;
    • replace() — завантажує сторінку із зазначеною у параметрі Url адресою й заміняє Url-адресу поточної сторінки (location.href).
